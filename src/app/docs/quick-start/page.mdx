# Quick Start Guide

Get up and running with Invalend Protocol in just 5 minutes. This guide will walk you through connecting your wallet, getting test tokens, and making your first transaction.

## Prerequisites

- MetaMask or compatible wallet
- Access to Lisk Sepolia testnet
- Basic understanding of DeFi concepts

## Step 1: Add Lisk Sepolia Network

First, add the Lisk Sepolia testnet to your wallet:

```json
{
  "chainId": "0x106a",
  "chainName": "Lisk Sepolia Testnet", 
  "rpcUrls": ["https://rpc.sepolia-api.lisk.com"],
  "nativeCurrency": {
    "name": "Sepolia Ether",
    "symbol": "ETH",
    "decimals": 18
  },
  "blockExplorerUrls": ["https://sepolia-blockscout.lisk.com"]
}
```

## Step 2: Get Test ETH

Get test ETH for gas fees from the [Lisk Sepolia Faucet](https://sepolia-faucet.lisk.com/).

## Step 3: Connect to Invalend

1. Visit the [Invalend dApp](/)
2. Click "Connect Wallet" in the top right
3. Select your wallet provider
4. Approve the connection

## Step 4: Get Test USDC

1. Navigate to the **Faucet** tab
2. Click "Mint 1000 USDC"
3. Confirm the transaction
4. Wait for confirmation (usually 1-2 minutes)

## Step 5: Choose Your Path

### Path A: Earn Yield (Recommended for beginners)

**Deposit USDC and earn 6% APY:**

1. Go to the **Earn** tab
2. Select "Deposit" 
3. Enter amount (start with 100 USDC)
4. Click "Approve USDC" (first time only)
5. Click "Deposit" 
6. Confirm transactions

**Result**: You're now earning 6% APY on your deposit!

### Path B: Leveraged Trading (Advanced)

**Borrow and trade with leverage:**

1. Go to the **Borrow** tab
2. Select "Borrow"
3. Enter loan amount (max 5x your collateral)
4. Approve USDC for collateral
5. Create loan
6. Use **Trade** tab to swap tokens
7. Monitor position in **Wallet** tab

## Step 6: Monitor Your Positions

Use the **Dashboard** to track:
- Lending positions and earnings
- Active loans and P&L
- Portfolio overview
- Transaction history

## Common First Actions

### Scenario 1: Conservative Investor
```
1. Mint 1000 USDC from faucet
2. Deposit 500 USDC to earn 6% APY
3. Keep 500 USDC for future opportunities
```

### Scenario 2: Active Trader

## Troubleshooting

### Common Issues

**Wallet won't connect**
- Refresh page and try again
- Check if wallet is on Lisk Sepolia network
- Clear browser cache

**Transaction fails**
- Ensure sufficient ETH for gas
- Check network congestion
- Try increasing gas limit

**Can't see balance**
- Wait for block confirmation
- Refresh the page
- Check transaction on block explorer

**Need help?**
